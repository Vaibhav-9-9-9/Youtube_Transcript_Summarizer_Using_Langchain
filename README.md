<h1 align="center">🚀 YouTube Transcript Summarizer using LangChain</h1>

<p align="center">
Convert <b>YouTube videos into concise summaries</b> using <b>LangChain + LLMs</b>

<hr>

<h2>📌 Overview</h2>
<p>
This project is an AI-powered application that extracts transcripts from YouTube videos 
and generates meaningful summaries using Large Language Models.
</p>

<p>
It helps users quickly understand long videos without watching them completely.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
<li>🎥 Paste any YouTube video URL</li>
<li>📝 Automatic transcript extraction</li>
<li>✂️ Handles long videos using text chunking</li>
<li>🤖 AI-powered summarization (LangChain + LLM)</li>
<li>⚡ Fast and simple Streamlit UI</li>
</ul>

<hr>

<h2>🧠 How It Works</h2>
<ol>
<li><b>Input:</b> User provides a YouTube URL</li>
<li><b>Transcript Extraction:</b> Fetches captions using API</li>
<li><b>Chunking:</b> Splits large text into smaller parts</li>
<li><b>LLM Processing:</b> Generates summaries</li>
<li><b>Output:</b> Displays final summarized content</li>
</ol>
